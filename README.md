# sireco
Sistema de relevamiento al contribuyente.
Es una pagina web simple desarrollada en html, css, javascript y mysql.
Se puede realizar una busqueda en la base de datos por medio del dni y completar el formulario con los datos del contribuyente en caso de que aun no fue cargado para que queden guardados en la base de datos.
